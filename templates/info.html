SOBRE EL PROYECTO

Este diagnosticador fue realizado como proyecto final del curso de Sistemas Inteligentes 2022-II tomando los conceptos de lógica difusa con el propósito de inferir la probabilidad que tenga el usuario de padecer de COVID-19 de acuerdo a los síntomas que ingrese.


¿Quiénes puede usar este diagnosticador?
  • Cualquier usuario puede usar esta webapp como referencia para diagnosticar un posible caso de contagio por COVID-19

¿Puedo usarlo en un celular?
  • Por supuesto, al ser una aplicativo web puede ser usado tanto en computadoras y dispositivos móviles a través de una navegador web (Chrome, Opera...)

METODOLOGÍA BASADA EN LA INFERENCIA DE MAMDANI

      I. Declaración de variables o síntomas
      II. Declaración de funciones de membresía por cada síntoma
      III. Declaración de reglas difusas o ruleset
      IV. Fuzzificación
      V. Evaluación de reglas difusas
      VI. Defuzzificación
